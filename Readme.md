# vigraha

cut Sanskrit samasa into possible chains - padacCeda

## Installation

With node.js:

````bash
$ npm install vigraha
````

## API

````javascript
var vigraha = require('vigraha');
var chains = vigraha.pdchs(samasa);
console.log(chains);
````


## CLI

````javascript
$ node run vihAya
or if you like
$ node run विहाय
````



## Running node tests

````bash
$ make gita
````
